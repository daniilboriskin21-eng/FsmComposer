# FSM Composer & Debugger

**Паттерны проектирования программного обеспечения**  
**Вариант 3**: *Абстрактная фабрика → Компоновщик → Команда*

---

## Описание

WPF-приложение для **создания**, **композиции** и **отладки** **конечных автоматов** (DFA / NFA).

### Функции
- Создание **DFA** и **NFA** через **Abstract Factory**
- **Композиция** FSM в иерархию (**Composite**)
- **Шаговая отладка** с **отменой** (**Command + Undo**)
- **Визуализация графа** (состояния, переходы, ε-переходы)
- Отображение текущего состояния и результата

---

## Паттерны

| Паттерн | Где | Что делает |
|-------|-----|-----------|
| **Abstract Factory** | `DfaFactory`, `NfaFactory` | Создание разных типов FSM |
| **Composite** | `AtomicFsm`, `CompositeFsm` | Иерархия простых и составных автоматов |
| **Command** | `StepCommand`, `CommandManager` | Шаг с поддержкой Undo |

---

## Как запустить

1. Открой `FsmComposer.sln` в **Visual Studio 2022+**
2. Убедись, что **.NET 9.0 SDK** установлен
3. Нажми **F5**
